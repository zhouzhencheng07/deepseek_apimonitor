# 悬浮球模式

## 目标
平时显示为透明悬浮球（置顶、只显示核心指标），点击展开为完整主界面。

## 改动

### 1. 窗口配置 `src-tauri/tauri.conf.json`

```json
{
  "decorations": false,    // 无边框
  "transparent": true,     // 透明背景
  "alwaysOnTop": true,     // 置顶
  "skipTaskbar": true,     // 跳过任务栏
  "width": 170,
  "height": 72
}
```

Tauri v2 需要确认 `transparent` 是否需要在 Rust 侧额外开启（`macos-private-api` 等）。

### 2. 前端双模式 `frontend/src/Dashboard.vue`

**球模式**（默认）：
- 透明背景，3 行显示：
  - 第1行：余额（`¥12.34`）
  - 第2行：日消费 | 日Token（`¥0.56  |  1,234`）
  - 第3行：命中率 | 输出（`85.2%  |  567`）
- `data-tauri-drag-region` 可拖动

**展开模式**：
- 点击球切换到当前主界面（汇总 + 模型 + 按日统计）
- 点击「收起」或「← 返回」切回球模式

**切换方式**：
- 点击球 ↔ 展开
- 窗口大小随之变化
- 用 Tauri API 调整窗口尺寸

### 3. 去掉 body 背景色

```css
html, body, #app {
  background-color: transparent !important;
}
```

## 涉及文件

| 文件 | 改动 |
|------|------|
| `src-tauri/tauri.conf.json` | 窗口透明、无边框、置顶 |
| `frontend/src/Dashboard.vue` | 双模式切换、球模式 UI |
| `frontend/src/App.vue` | 去掉背景色 |
| `src-tauri/src/main.rs` (可能) | 如果 v2 需要额外配置透明 |
