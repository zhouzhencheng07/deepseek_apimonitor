# 悬浮球独立窗口方案

## 思路

悬浮球是一个**独立的 Tauri 窗口**，跟主界面窗口无关。主界面不动。

## 改动

### 1. Dashboard.vue
- 加一个「悬浮球」按钮
- 点击按钮创建新窗口（小、无边框、置顶）
- 主界面本身不变

### 2. BallView.vue（新增）
悬浮球窗口的专属组件，只显示 3 行数据：
```
¥12.34
¥0.56 | 1,234
85.2% | 567
```
- 无边框、透明背景
- data-tauri-drag-region 可拖动
- 点击关闭（或聚焦主界面）

### 3. main.js
根据 URL 参数 `?ball` 决定渲染哪个组件：
- 有 `?ball` → 渲染 BallView
- 没有 → 渲染 Dashboard（原主界面）

### 4. 权限
- 创建窗口权限
- 关闭窗口权限
- 拖拽权限

## 涉及文件

| 文件 | 改动 |
|------|------|
| `frontend/src/Dashboard.vue` | 加「悬浮球」按钮 + 调用创建窗口 |
| `frontend/src/BallView.vue` | 新建，球 UI |
| `frontend/src/main.js` | 根据 URL 参数切换组件 |
| `src-tauri/capabilities/default.json` | 加窗口权限 |

## 优势
- 主界面完全不动
- 两个窗口互不干扰
- 窗口配置各自独立
